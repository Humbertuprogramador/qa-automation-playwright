# QA Automation Project - Playwright

## 📌 Descripción
Proyecto de automatización QA utilizando Playwright para validar funcionalidades UI y API sobre una aplicación real.

## 🚀 Tecnologías
- JavaScript
- Playwright
- Node.js

## 🧪 Casos de prueba
- Login exitoso
- Login fallido
- Flujo de compra (ecommerce)
- Test de API (GET users)

## 🧱 Arquitectura
- Page Object Model (POM)
- Separación de tests UI y API
- Uso de selectores robustos
